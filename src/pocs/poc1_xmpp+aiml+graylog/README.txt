What is this?
 - First proof of concept of AIML chatbot + XMPP Client connecting to Facebook or gmail Chat + Graylog events

Dependencies:
 * Qt 4.7
 * QXmpp (update SadoskyPoc1.pro with the path of QXmpp)
 * zlib

How to run?
 1. First edit config.ini and set your username and password to connecto to gmail or facebook
 2. Run ./SadoskyPoC1
